Check This List will create checklists that need to be met before publishing a new post

### Description

Set up your own custom checklist of tasks or expectation that need to be met, save them and as long as this plugin is active, you will have to check off each task before being able to publish a new post.

This is designed to help remind you of simple things like inserting an image or spellchecking. 

### Installation

1. Upload `check-this-list` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
